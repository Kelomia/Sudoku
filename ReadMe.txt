This project is the Sudoku.
User can load Sudoku's value by file "Sudoku.txt", it will read and fill board.
The Read progress will stop when meet Signal "#", 1-9 mean 1-9, 0 mean blank.

It can check if the game is completed. And output message.
The error blank will print in command-windows, which is not fit in Javafx output window yet.

If the "Sudoku.txt" file has invalid value, not [0-9] and '#', it will print out the line where
the invalid value is.

Auto-complete function is in work.
